# mastermind-embedded-game
Embedded Mastermind game implemented in C and ARM Assembly on Raspberry Pi, featuring GPIO control, LCD display, LEDs, push-button input, and unit testing.

## Features

- Mastermind game implementation
- ARM Assembly matching algorithm
- GPIO communication
- LCD display support
- LED indicators and push-button input
- Unit testing
- Makefile build system

## Technologies Used

- C
- ARM Assembly
- Raspberry Pi
- Linux
- GPIO Programming
- Makefile

## Project Structure


master-mind.c    Main game logic
lcdBinary.c      LCD and GPIO interface
mm-matches.s     ARM Assembly implementation
testm.c          Unit tests
test.sh          Test script
Makefile         Build configuration

## Build

```bash
make all
```

Run:

```bash
make run
```

Test:

```bash
make test
```
